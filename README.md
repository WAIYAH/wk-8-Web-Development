# Fresh Foods & Vegetables - E-commerce Website

## Project Overview
This project is a fully functional e-commerce website for "Fresh Foods & Vegetables," designed to sell organic produce online. It demonstrates proficiency in web development using HTML5, CSS3, JavaScript, and Bootstrap. The site features a clean, responsive design with a yellow (#FFD700) and black (#000000) theme, smooth scrolling, animations, and interactivity such as a product grid, cart functionality, and form validation.

## Features
Multi-Page Structure: Four pages (Home, Shop, About, Contact).
Responsive Design: Built with Bootstrap 5 for mobile-first responsiveness.

## Interactive Elements:
Product grid with filtering and sorting (Shop page).
Add-to-cart functionality with local storage.
Newsletter subscription with email validation.
Contact form with client-side validation.


Animations: Smooth transitions and effects using Animate.css.
SEO Optimization: Includes meta tags for search engine and social media sharing.
Deployment Ready: Deployable on GitHub Pages, Netlify, or Vercel.

## Technologies Used

HTML5: For semantic structure.
CSS3: Custom styling with smooth scrolling and animations.
JavaScript: For interactivity (cart, filtering, form validation).
Bootstrap 5.3.3: For responsive layout and components.
Animate.css: For animations.
Bootstrap Icons: For icons in the cart and navigation.

## Deployment
The website is deployed on GitHub Pages:

Live URL: https://wk-8-web-development.vercel.app/
To deploy:
Push all files to your GitHub repository.
Go to Settings > Pages in your GitHub repository.
Set the source to the main branch and / (root) folder.
Save, and GitHub Pages will provide a live URL.



Alternatively, you can deploy on Netlify or Vercel:

Netlify: Drag and drop the project folder into Netlify's dashboard.
Vercel: Import the repository via Vercel's dashboard.

## Usage

Home Page (index.html): Features a hero section, featured products, and newsletter subscription.
Shop Page (shop.html): Browse products with category filtering and price sorting. Add items to the cart.
About Page (about.html): Learn about the company, mission, and team.
Contact Page (contact.html): Submit inquiries via a validated contact form.
Cart: Add products to the cart (stored in local storage) and view the cart count in the header.
